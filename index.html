Font Style Cheat Sheet

Font Properties

Property Name Description
font-family
 The font family The font-family property specifies a list of fonts, from highest priority to lowest.
font-size
 The font size The font-size CSS property sets the size of the font.
font-weight
 The font weight The font-weight CSS property sets the weight (or boldness) of the font.
font-style
 The font style The font-style CSS property sets whether a font should be styled with a normal,
italic, or oblique face from its font-family.
font-weight
 The font weight The font-weight CSS property sets the weight (or boldness) of the font.
